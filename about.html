Hi, I'm Michael Wang.

I'm a college student that wants the change the world before I leave it.

What does "I do not code, I am just a hacker" mean?

The phrase is a reference to Childish Gambino's (Donald Glover's) mixtape and motif, "I do not talk, I am just a rapper." A lot of us spend time talking without saying anything particularly meaningful, and I believe the same can be said about programming.

This short phrase is neatly packages my personal rebellion to becoming one of the things I fear most: a code monkey. I want to create meaningful software that touches people's lives, not write meaningless code to get a paycheck. I'm looking for work that I can be excited about.

Of course I can write code (millions of people can and do it very well), but at the end of the day if the code I write isn't useful or written with purpose, then I can't see the point anyways.

TL;DR I'm young and idealistic and want to be different.
